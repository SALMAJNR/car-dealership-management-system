# 🚗 Car Dealership Management System

A full-stack web application built with Django REST Framework and Vue.js for managing cars, customers, dealers, brands, and categories.

---

## 📖 Overview

This project was developed as part of a university software development course and enhanced as a personal portfolio project. It demonstrates backend API development with Django REST Framework and frontend development with Vue.js.

---

## ✨ Features

- User Authentication
- Car Management
- Customer Management
- Dealer Management
- Brand Management
- Category Management
- REST API
- Vue.js Frontend
- CRUD Operations

## 🚀 Getting Started

### Backend

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend

```bash
cd client
npm install
npm run dev
```
